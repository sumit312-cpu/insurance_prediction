# Insurance Cost Prediction Model

## 📌 Project Overview
This project aims to predict insurance charges based on customer attributes such as age, BMI, smoking status, and region using Machine Learning techniques.

## 📊 Dataset Features
- Age
- Sex
- BMI
- Children
- Smoker
- Region

## ⚙️ Steps Involved
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building
6. Model Evaluation

## 🤖 Models Used
- Linear Regression
- Decision Tree
- Random Forest

## 📈 Results
- Best Model: Random Forest
- Accuracy: 87%

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 💡 Key Insights
- Smoking significantly increases insurance cost
- BMI and age are strong predictors
